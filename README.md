# Snake Game 🐍

A classic snake game I built from scratch using Python as part of my learning journey.

## How to run

Make sure you have Python installed, then run:

python main.py

Use the arrow keys to control the snake.
Eat the blue food to grow and score points.
Don't hit the walls or your own tail!

## What I built with

- Python
- Turtle graphics (built into Python, no installs needed)

## What I learned

- How to use classes and objects (OOP)
- How inheritance works
- Breaking a project into multiple files
- Building a game loop
